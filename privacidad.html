Política de Privacidad

Última actualización: 22 de junio de 2026

1. Introducción

Esta política de privacidad explica cómo se maneja la información de los visitantes en este sitio web (publicalex.github.io) y en las aplicaciones publicadas por Alexander Aragón.

2. Datos que recopilamos

Este sitio web es un portafolio estático. No recopilamos datos personales de los visitantes. No se utilizan cookies de rastreo, formularios de registro ni herramientas de análisis que almacenen información del usuario.

El único dato que podría recopilarse es la dirección IP, utilizada únicamente por los servidores de GitHub Pages para la entrega del contenido, siguiendo sus propias políticas de privacidad.

3. Servicios de terceros

Este sitio utiliza los siguientes servicios de terceros:

  • Google Fonts — para la carga de tipografías. Al cargarse, Google puede recibir tu dirección IP.
  • Tailwind CSS CDN — para los estilos de la interfaz.
  • GitHub Pages — para el alojamiento del sitio.
