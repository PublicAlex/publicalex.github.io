Política de Privacidad

Última actualización: 22 de junio de 2026

1. Introducción

Esta política de privacidad describe cómo AlertaDestino ("la aplicación", "nosotros") recopila, usa y protege la información de los usuarios. Al utilizar la aplicación, aceptas las prácticas descritas en este documento.

2. Datos que recopilamos

2.1 Ubicación GPS (Datos de localización)

La aplicación accede a la ubicación precisa del dispositivo (GPS) para su funcionalidad principal: monitorear la posición del usuario y activar una alarma cuando este se encuentra dentro de la zona geográfica (geocerca) que ha configurado como destino.

  • Se accede a la ubicación en primer plano y en segundo plano (background), incluso cuando la aplicación está cerrada, para garantizar que la alarma se active correctamente.
  • Los datos de ubicación se procesan en tiempo real únicamente para calcular la distancia al destino configurado.
  • Se mantiene un registro temporal en memoria de los últimos puntos de ubicación (máximo 200) para funcionalidades de visualización de ruta. Estos datos se eliminan al detener el seguimiento.

2.2 Datos de configuración

La aplicación almacena localmente en el dispositivo la siguiente información:

  • Nombre, latitud y longitud del destino configurado.
  • Historial de destinos guardados (hasta 20 destinos).
  • Preferencias de configuración: radio de geocerca, tiempo de espera por pérdida de señal, volumen de alarma, umbral de precisión GPS.
  • Estado del seguimiento (activo/inactivo).

2.3 Consultas de búsqueda

Cuando utilizas el buscador de lugares, el texto de búsqueda se envía al servicio Nominatim de OpenStreetMap para obtener resultados de geolocalización. No almacenamos ni registramos estas consultas.

3. Cómo usamos los datos

Los datos de ubicación se utilizan exclusivamente para:

  • Calcular la distancia entre tu posición actual y el destino configurado.
  • Activar la alarma cuando ingresas a la geocerca del destino.
  • Detectar anomalías en la señal GPS (saltos de ubicación imposibles) y activar alarmas de respaldo.
  • Detectar pérdida prolongada de señal GPS y activar la alarma de seguridad correspondiente.
  • Mostrar tu posición actual y ruta en el mapa (OpenStreetMap).

4. Almacenamiento y transferencia de datos

Todos los datos de ubicación y configuración se almacenan exclusivamente en el dispositivo del usuario mediante el sistema de almacenamiento local de Android (DataStore).

No recopilamos, transmitimos ni compartimos ningún dato de ubicación con servidores externos controlados por el desarrollador. La aplicación no envía información de ubicación a ninguna API o servicio de terceros para su procesamiento.

La aplicación requiere conexión a internet únicamente para:

  • Cargar los mapas desde CartoDB (servicio de mapas en línea).
  • Realizar búsquedas de lugares a través de Nominatim/OpenStreetMap.

Ninguno de estos servicios recibe datos de ubicación del usuario; solo se envían las consultas de búsqueda necesarias para su funcionamiento y las solicitudes de teselas de mapa.

6. Retención de datos

Los datos de ubicación se procesan en tiempo real y no se conservan después de detener el seguimiento. Los datos de configuración (destinos guardados, preferencias) permanecen almacenados en el dispositivo hasta que el usuario los elimine manualmente o desinstale la aplicación.

7. Control del usuario

El usuario tiene control total sobre sus datos:

  • Puede iniciar y detener el seguimiento en cualquier momento desde la pantalla principal.
  • Puede eliminar destinos guardados individualmente desde la lista de destinos.
  • Puede desinstalar la aplicación para eliminar todos los datos almacenados localmente.
  • Puede revocar los permisos de ubicación desde la configuración del sistema Android en cualquier momento, aunque esto impedirá el funcionamiento de la aplicación.

8. Seguridad de los datos

Almacenamos los datos localmente utilizando el sistema de almacenamiento seguro de Android (DataStore). Los datos no se transmiten a través de la red ni se almacenan en servidores externos.

9. Cambios en esta política de privacidad

Podemos actualizar esta política de privacidad periódicamente. Notificaremos cualquier cambio actualizando la fecha de "Última actualización" al inicio de este documento. Se recomienda revisar esta política periódicamente para estar informado sobre cómo protegemos tu información.
